🚀 Running a Local Server

Option 1: Python (if installed)

python -m http.server 8000

Then visit: http://localhost:8000

Option 2: Node.js

npx http-server

Or install globally: npm install -g http-server

Option 3: PHP

php -S localhost:8000

Option 4: VS Code

Install "Live Server" extension and click "Go Live" button

🔍 Common Issues & Solutions

Videos Not Playing

✅ Solution: Run from a local server (required for HLS.js)

✅ Check: Internet connection (videos stream from external URLs)

Mobile Menu Not Working

✅ Check: JavaScript is enabled

✅ Check: No console errors

Glass Effects Not Visible

✅ Check: Browser supports backdrop-filter (most modern browsers)

✅ Try: Chrome/Edge/Safari (best support)

Layout Issues

✅ Check: Viewport meta tag is present

✅ Clear: Browser cache and refresh
